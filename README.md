# VercelAISDK-Sample
Vercel AI SDK調査・検証用のリポジトリです。

## Vercel AI SDKの始め方

```bash
yarn create next-app@latest my-ai-app
yarn add ai @ai-sdk/openai zod
```

環境変数に open AIのAPIキーをセットすればOK!

```txt
OPENAI_API_KEY
```
