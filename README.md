# VercelAISDK-Sample

Vercel AI SDK 調査・検証用のリポジトリです。

## Vercel AI SDK の始め方

```bash
bun create next-app@latest my-ai-app
bun add ai @ai-sdk/openai zod
```

環境変数に open AI の API キーをセットすれば OK!

```txt
OPENAI_API_KEY
```
